# lite-llm

Independent Rust crate skeleton for Lite LLM.

## Purpose
This crate provides a minimal compile-ready starting point.

## Structure
- `src/lib.rs`: base public API scaffold

## Build
```bash
cargo check
```

## Notes
- Standalone crate (no cross-crate dependencies)
- Behavior is scaffolded, not production-complete
- License for this crate is in `LICENSE`
